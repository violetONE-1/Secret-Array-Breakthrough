密阵突围：
整个游戏界面是25*25的矩形网格，每个格子中随机填充一个字母和一个数字，同一盘游戏的初始界面应该是一致的，同时要保证不会出现全盘都无处可走的情况。玩家可以选择五个格作为自己的初始点，周围上下左右相邻的四个格子中，若有字母与中心相同而数字不同的，可以合并这两格，中心移动，空出位置；数字相同而字母不同的同理。并且合并后，对应不同的数字或者字母要相加。如A1与A2合并后变成A3，A1与B1合并后变成C1。数字超过一位数则取个位，字母超过Z则顺移到A往后。若数字与字母都相同则合并后保持不变。


赛题解析：
1)	选题：综艺节目《最强大脑》第13季《密阵突围》
2)	实现内容
Level 1：（必做）
1.	以合适的界面和交互方式呈现赛题
2.	模拟比赛要求，支持用户以玩家的方式做题，提交操作
3.	判断操作正误，并给出对应的提示和盘面变化
4.	内含3道以上的题面，支持用户选题作答
5.	记录玩家答题的信息，比如用时、正确率、得分等，形成用户排行榜

Level 2：（选做）
1.	图形化界面
2.	可设置盘面大小，自动出题
3.	支持用户自己创建题目并作答
4.	可将新题保存到题库
5.	形成多元的用户排行榜，比如单题最快，累计分数最高等
6.	保存与回放功能，将每回合移动序列保存到文本文件，实现“回放模式”，按步骤重现对局
7.	简单AI对手（AI可按规则操作）

Level 3：（选做）
1.	更强的图形化交互、动画和音效
2.	玩家答题过程中有提示
3.	高级AI对手，AI有更多智能体现，比如防守/进攻策略
4.	游戏以闯关等更丰富的形式进行
5.	其他任何围绕游戏主题的延申和扩展

3、根据问题进行学习，拓广、深化。

	1、每位同学独立完成一个小型应用程序的设计、编码和测试；
	2、程序要求用到面向对象编程的特性；
	3、程序源代行数要求： 1000行以上（自编代码，不包括引入的第三方库）；
	4、程序必须包括文件操作，数据存储不允许使用数据库。

6、题目：最强大脑赛题《密阵突围》。 
7、开发语言必须是C++，开发工具不限，常用的如VC，QT等；
8、用于个人电脑的应用，不包括手机应用和网站制作。
9、鼓励使用大模型辅助开发，但要保持作品的原创性，以个人学习和能力提升为使用目标。
